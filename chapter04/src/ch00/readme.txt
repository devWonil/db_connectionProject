서버랑 클라이언트 아무거나 실행 시켜도 오류 안남.
사용자한테 보여줘야하는 부분은 public
불필요하면 그 외 전부 private


<Server>

서버 포트 입력하고 서버 실행하면 메시지 출력 
서버 중지 누르면 메시지 출력하고 서버 실행 버튼 활성화
서버를 실행한 상태라면 서버 중지 버튼 비활성화
서버 실행 버튼과 서버 중지 버튼은 항상 하나만 활성화



<Client>

로그인 화면
HOST_IP 고정

서버가 사용자의 접속을 기다림..
서버와 포트번호 같게 입력하고 connect하면 서버에 메시지 출력
연결된 상태가 아니라면 방만들기 버튼 클릭 시 오류

연결되면 방 만들기
방 이름 입력하면 대기실 화면에 해당 아이디로 방 만들어짐
방 만들었다면 방 만들기 기능이 비활성화 됨
로그인 화면에서 방을 나갈 수 있음
아이디 하나 당 방 하나 만들 수 있음
방 만들기와 방 나가기 기능은 로그인, 대기실, 채팅 화면에서 전부 가능


쪽지보내기 -> 오류있음
대상을 선택하라고 나오지만 쪽지가 null로 보내지는 오류
접속자를 클릭하여 쪽지를 보내면 잘 보내짐

퇴장하면 퇴장 출력하고 채팅 전송

멀티 쓰레딩
